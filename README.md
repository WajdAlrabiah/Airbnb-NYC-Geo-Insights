# Airbnb-NYC-Geo-Insights
An exploratory data analysis and geospatial visualization project based on New York City Airbnb data from Kaggle. The project uses Pandas and GeoPandas to clean, transform, and analyze the dataset, uncovering trends in pricing, availability, and neighborhood activity.

Interactive dashboards and maps were created using Plotly, Folium, and Streamlit to visualize spatial patterns, identify high-demand areas, and highlight luxury listings.

Key Features:

Data cleaning, handling missing values, and feature engineering.

Visualization of price distributions, correlations, and neighborhood comparisons.

Interactive geospatial dashboards showing heatmaps and listing densities.

Insights into host activity, listing types, and neighborhood popularity.

Tools & Libraries:
Python, Pandas, NumPy, GeoPandas, Matplotlib, Plotly, Folium, Streamlit, KaggleHub
